# 🩺 Diabetes Prediction App

A simple and interactive **Machine Learning web app** built with **Streamlit** that predicts whether a person is diabetic or not based on medical input parameters.

---

## 🚀 Features

* 🔍 Predicts diabetes using **SVM (Linear Kernel)**
* 📊 Displays **model accuracy**
* 🧮 Built-in **BMI Calculator** (for users who don’t know their BMI)
* ⚡ Fast and responsive UI
* 🎯 Clean and compact layout (screenshot-friendly)

---

## ⚠️ Important Note

> This model is trained on a dataset consisting of **female patients only**.
> Predictions may **not be accurate for males**.

---

## 📂 Dataset

* File: `diabetes.csv`
* Commonly known as the **Pima Indians Diabetes Dataset**
* Features include:

  * Pregnancies
  * Glucose
  * Blood Pressure
  * Skin Thickness
  * Insulin
  * BMI
  * Diabetes Pedigree Function (DPF)
  * Age

## 🖥️ Usage

1. Enter patient details:

   * Glucose level
   * Blood pressure
   * Age, etc.
2. Choose:

   * Enter BMI manually
     **or**
   * Calculate BMI using height & weight
3. Click **Predict**
4. Get instant result:

   * ✅ Non-Diabetic
   * ⚠️ Diabetic
